Простое приложение фонарик для Android устройств.

## Требования

- Android SDK 21+ (Android 5.0 Lollipop и выше)
- Android Studio Arctic Fox или новее
- Устройство с камерой и вспышкой

## Функционал

- Включение/выключение вспышки камеры одним нажатием
- Простой и интуитивный интерфейс
- Автоматическое выключение фонарика при выходе из приложения
- Запрос разрешений на использование камеры

## Установка

1. Откройте проект в Android Studio
2. Дождитесь завершения синхронизации Gradle
3. Подключите Android устройство или запустите эмулятор
4. Нажмите кнопку "Run" (▶️) для установки и запуска приложения

## Использование

1. При первом запуске предоставьте разрешение на доступ к камере
2. Нажмите на большую круглую кнопку в центре экрана
3. Фонарик включится/выключится

## Примечания

- Приложение использует Camera2 API для управления вспышкой
- Для работы требуется устройство с камерой, оснащенной вспышкой
- Фонарик автоматически выключается при закрытии приложения
